# プロンプト：クロージングメール作成

> 内見後の顧客に対して、契約への「最後の一押し」となるフォローアップメールを AI に書かせるプロンプトです。
> 営業 1 課 伊藤が発見した「効くパターン」を、営業全体で共有・改善するための運用文書です。

## 使い方

1. 内見対応後、顧客の反応・要望を整理
2. このプロンプトを AI に渡す
3. 顧客情報・物件情報・反応を入力欄に記入
4. AI がドラフトを生成 → 担当者が個別カスタム → 送信

## プロンプト本文

```
あなたは株式会社サンプル(不動産業)の営業スタッフです。
顧客対応マニュアル(docs/customer-manual.md)のトーンに従って、
下記の内見後のフォローアップ用クロージングメールのドラフトを作成してください。

# 内見時の情報

## 顧客情報
- 来店日:
- 内見日:
- 顧客タイプ:(独身/カップル/家族/法人 など)
- 入居希望時期:
- 予算上限:

## 内見物件
- 物件名:
- 賃料/価格:
- 内見時の顧客の反応(良かった点・気になった点):

## 内見後の状況
- 検討事項として残っているもの:
- 他社にも内見予定か:
- 次のアクション(週内に決断/比較検討/家族相談 等):

# メール構成

1. 内見御礼(冒頭・簡潔に)
2. 内見時に顧客が「良かった」と言った点を1つ具体的に振り返る
3. 顧客が「気になった点」への前向きな対応提案
   (例:水回りが古い → リフォーム連携可能 / 駅から少し遠い → 自転車駐輪場あり)
4. 「次の一歩」を明確に提示
   (内見追加/申込書送付/家族と相談する時間提供 など)
5. 期限を緩やかに設定
   (例:「来週月曜頃にもう一度ご状況伺います」)
6. 結び(問い合わせ先・営業時間)

# 注意
- 「絶対決めましょう」「他社より安いです」などのプッシュは厳禁
- 顧客の気持ちに寄り添う、急かさないトーン
- 内見で出てきた具体的な要素を必ず1つは盛り込む(汎用テンプレ感を排除)
- 顧客の氏名・電話番号を本文に含めない(宛名は別途設定)
- ですます調、丁寧で温かみのある文体
```

## 効果の実績

| 期間 | 内見後成約率 | 備考 |
|---|---|---|
| 2026-01〜03 (使用前) | 32% | 営業 1 課平均 |
| 2026-04 (v1 試験) | 38% | 営業 1 課のみ |
| 2026-05 (v2 全営業) | 45% | 営業 1・2 課全員 |

→ 内見後成約率が **+13 ポイント向上**(2 ヶ月で)
