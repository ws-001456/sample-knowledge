# 経営方針 2026

> 株式会社サンプルの 2026 年度経営方針を定める文書です。
> 全社員および AI(Claude Code 等)が、当社の方向性を理解するための基準となります。

## ビジョン

**「地域に深く根ざした、長く信頼される不動産パートナーになる」**

短期の売上ではなく、3 世代にわたる顧客関係を築くことを最優先する。

## 2026 年度の重点方針

### 1. 管理物件 500 件達成

- 2025 年度末：380 件
- 2026 年度末目標：**500 件**
- 達成手段：地主層への定期訪問・既存オーナーからの紹介経由を強化

### 2. AI 活用による業務効率化

- 重要事項説明書のドラフト生成を AI で半自動化
- 物件マッチング AI の試験導入
- スタッフ全員が GitHub にナレッジを集約し、AI が読める状態に

### 3. 若手スタッフの育成

- 新人オンボーディング期間を 3 ヶ月 → 2 ヶ月に短縮
- 業務マニュアルを GitHub Wiki に集約
- ベテランの暗黙知を文書化(月 1 件以上)

## 利益方針

- 売上目標：8.2 億円
- 営業利益率：12%
- 営業利益：9,840 万円
