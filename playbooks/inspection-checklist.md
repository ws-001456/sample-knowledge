# 物件視察チェックリスト

> 仲介物件の現地視察時に確認すべき項目をまとめたチェックリストです。

## 視察前(オフィスで準備)

- [ ] 物件資料の最新版を印刷 or タブレットに用意
- [ ] レーザー距離計(ある場合)
- [ ] スマートフォン(写真撮影用)
- [ ] 名刺(オーナーや管理会社と接触する可能性)
- [ ] メジャー(間取り確認用)

## 建物外観(到着 0〜10 分)

- [ ] 駅からの徒歩時間を実測
- [ ] 周辺環境(コンビニ・スーパー・治安)
- [ ] 騒音(隣接道路・線路・工場)
- [ ] 駐輪場・駐車場の状況
- [ ] エントランス・郵便受けの管理状態
- [ ] エレベーター(あれば)の管理状態

## 専有部分(10〜40 分)

- [ ] 専有面積(資料記載と実測の照合)
- [ ] 天井高
- [ ] 採光(各部屋の明るさ・方角)
- [ ] 風通し
- [ ] 床(きしみ・傾き)
- [ ] 壁紙の状態(汚れ・破れ)
- [ ] 水回り(キッチン・浴室・トイレ・洗面)
- [ ] エアコン・給湯器の動作確認(可能なら)
- [ ] コンセント数と位置
- [ ] 収納のサイズ・湿気
- [ ] バルコニーの広さ・洗濯物干しスペース

## インフラ(40〜50 分)

- [ ] インターネット環境(光回線対応か)
- [ ] ガス(都市ガス/プロパン)
- [ ] 携帯電話の電波状況
- [ ] 宅配ボックス
- [ ] オートロック・防犯カメラ

## 視察後(50〜60 分)

- [ ] 写真整理(物件単位でフォルダ化)
- [ ] チェックリストを社内システムに登録
- [ ] 気になった点を 3 行でメモ
- [ ] 顧客への案内可否を判断
